Button Component

The Button component provides a reusable way to trigger actions within your application. It supports multiple variants, sizes, and accessibility features.

Usage Example

          
import { Button } from '@components/Button';

Props

Prop Type Default Description
variant 'primary' | 'secondary' 'primary' Controls the color and style of the button.
onClick () => void — Callback fired when the button is clicked.
